Introduction

This assignment is an expansion of the bakery website I made eariler.
Requirements for this assignment involve using React to add new functionality, such as filtering items based on different qualities.

Background

My previous bakery e-commerce website was usable and functional, but it did not look visually appealing.

I planned on making much-needed visual changes in addition to adding the required sorting and filtering functions for this assignment. The new website needed to be able to allow the user to filter the bakery items for specific qualities (e.g. French items), as well as sort the items based on a numeric metric like price.

Implementation

I used the same frameworks (React, CSS) as the React studio. For this assignment, I used Material UI components to implement the sorting and filtering functions.
For filtering, I used Material UI checkboxes so the user can choose the filters applied. I included a Radio button set to sort the bakery items by price. In the future, I could add more buttons so users can sort items using different metrics.

I used CSS Grid to display the bakery items in an organized way.
